# 场景：创意与内容制作

## 角色画像
- 设计师、视频编辑、内容创作者。
- 需要素材管理、灵感记录、跨设备切换。

## 推荐配置
| 模块 | 作用 |
| --- | --- |
| Asset Board | 管理素材、自动生成标签 |
| Color Studio | 全局取色与调色板同步 |
| Storyboard | 快速搭建脚本大纲 |
| Media Preview | 直接在 CoreBox 浏览音视频 |

## 工作流示例
1. 拖入图片即触发「压缩 → 上传 → 生成分享链接」。
2. 使用 Clipbox 将灵感碎片转成待办，并自动发送到 Drafts。
3. 多设备间通过 Workspace 同步色板与字体配置。

## 灵感保护
- 启用「灵感模式」后自动保存剪贴板，避免丢失。
- 使用 Notifications Widget 查看反馈与互动实时状态。
