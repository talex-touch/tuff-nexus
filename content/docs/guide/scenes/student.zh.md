# 场景：学习与考试

## 角色画像
- 大学生、考研生、自主学习者。
- 同时管理课程资料、网课、备忘与倒计时。

## 推荐配置
| 模块 | 用途 |
| --- | --- |
| Clipbox + OCR | 批量整理笔记截图，键入即可检索 |
| Task Capsule | 课程/自习任务按优先级显示 |
| Pomodoro Widget | 结合 CoreBox 快捷控制专注节奏 |

## 操作流示例
1. 在 Workspace 中创建「学习」空间，开启沉浸主题。
2. 使用 CoreBox 输入 `note new` 开始速记，对应内容自动同步至云端。
3. 通过智能工作流将截屏 → OCR → 任务创建串成闭环，避免重复劳动。

## 小技巧
- 将每日倒计时 widget 固定在桌面，随时看到目标日期。
- 通过 `Study Sync` workflow 自动整理老师发的压缩包到课程文件夹。
