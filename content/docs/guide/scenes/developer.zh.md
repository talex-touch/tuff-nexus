# 场景：研发与自动化

## 角色画像
- 后端/前端工程师、自动化工程师、独立开发者。
- 需要在单屏上快速切换代码、终端、监控与文档。

## 推荐配置
| 模块 | 功能 |
| --- | --- |
| Git Watcher | 监听仓库变动，自动推送提醒 |
| API Tester | 在 CoreBox 内直接发起请求 |
| Snippet Vault | 片段收藏 + 智能补全 |
| Terminal Capsule | 原生终端 + 插件交互 |

## 工作流示例
1. 在 Workflow 中配置「CI 事件 → 通知 → 打开日志窗」。
2. 使用 Channel API 将本地脚本输出推送到 Widget，实时查看指标。
3. 通过智能工作流将需求工单同步到 Task Capsule，并维护状态。

## 增强体验
- 启用「代码模式」主题与等宽字体。
- 将常用仓库 pin 到 CoreBox，输入项目名即可打开。
- 利用多 Workspace 将个人项目与工作项目隔离，避免通知打扰。
