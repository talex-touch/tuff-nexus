# Scene: Creative Work

## Persona
- Designers, video editors, content creators.
- Manage assets, capture inspiration, swap devices mid-stream.

## Setup
| Module | Purpose |
| --- | --- |
| Asset Board | Catalog assets with auto tags |
| Color Studio | Global eyedropper + palette sync |
| Storyboard | Draft scripts and shot lists |
| Media Preview | Browse AV files inside CoreBox |

## Flow
1. Drop an image to trigger “compress → upload → share link”.
2. Route Clipbox ideas into Drafts via workflows.
3. Sync palettes and fonts across devices through workspace settings.

## Safeguards
- Enable Inspiration Mode to auto-save clipboard entries.
- Notifications widget surfaces engagement stats in real time.
