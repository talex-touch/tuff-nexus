# Tuff 使用指南

> 一款为多任务桌面场景打造的智能触控工作台。

## 产品定位
- **核心理念**：以 CoreBox 为枢纽，把插件、快捷指令、同步能力纳入同一套体验。
- **用户画像**：需要跨设备、跨系统保持节奏的创作者、工程师与知识工作者。
- **设计原则**：极简界面、即时反馈、系统级安全隔离。

## 功能矩阵
| 维度 | 能力 | 亮点 |
| --- | --- | --- |
| 启动与搜索 | CoreBox、智能索引、模糊调度 | 毫秒级搜索、可自定义触发词 |
| 工作流 | Intelligence Workflow、快捷脚本 | 图形化拼装，高阶用户可落地 JS/TS 插件 |
| 内容管理 | Clipbox、文件抽屉、跨端同步 | 沙箱隔离 + 端到端加密同步 |
| 插件生态 | 官方精选 + 开源扩展 | Manifest 驱动，热重载调试 |
| 个性化 | Workspace、主题、布局 | 针对多屏、多账户的场景配置 |

## 新手路径
1. 阅读《[快速开始](./start.zh.md)》完成安装与账号配置。
2. 在《[插件生态](./features/plugin-ecosystem.zh.md)》了解可用功能包。
3. 根据身份跳转至《[场景指南](./scenes/developer.zh.md)》等篇章，复制推荐模板。

## 深度使用
- **效率技巧**：前往《/tips》章节，掌握智能工作流、剪贴板编排等隐藏能力。
- **共创插件**：从《/dev》章节开始，按 Manifest → API → 发布的顺序逐步扩展你的工具箱。

## 术语速览
- **CoreBox**：系统级启动器，支持多窗口与分屏定位。
- **Plugin Capsule**：插件运行沙箱，保证权限隔离与配额控制。
- **Workspace**：保存布局、快捷键映射与同步策略的空间配置。
