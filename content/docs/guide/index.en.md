# Tuff Guide

> A desktop command center built for multi-tasking humans.

## Positioning
- **Core idea**: CoreBox orchestrates plugins, shortcuts, and sync under one calm surface.
- **Audience**: Creators, engineers, and knowledge workers juggling devices.
- **Principles**: Minimal UI, instant feedback, system-grade isolation.

## Feature Matrix
| Pillar | Capability | Highlights |
| --- | --- | --- |
| Launch & Search | CoreBox, smart index, fuzzy routing | ms-level results, custom triggers |
| Workflows | Intelligence Workflow, quick scripts | Visual builder + JS/TS escape hatch |
| Content | Clipbox, file drawers, sync | Sandbox isolation + end-to-end encrypted sync |
| Plugins | Curated + open ecosystem | Manifest-driven, hot reload |
| Personalization | Workspace, themes, layouts | Multi-screen & multi-account aware |

## Glossary
- **CoreBox**: system-grade launcher with multi-window awareness.
- **Plugin Capsule**: sandboxed container with quota controls.
- **Workspace**: bundle of layouts, shortcuts, and sync policies.
