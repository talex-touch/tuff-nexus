# Workspace 与个性化

## 多空间理念
- 将不同任务（个人、团队、临时项目）拆分为独立 Workspace。
- 每个 Workspace 保存窗口布局、插件集合、同步策略与快捷键映射。

## 创建与切换
1. 打开设置 → Workspace → 点击「新建」。
2. 选择模板（空白、写作、开发、设计）或从现有空间复制。
3. 使用 Dock 边栏或 `⌘ + Shift + ↑/↓` 极速切换。

## 同步策略
| 模式 | 使用场景 | 说明 |
| --- | --- | --- |
| 实时 | 桌面 + 笔电同时作业 | 保存即同步，冲突以时间戳解决 |
| 智能 | 多设备偶尔切换 | 在 idle 或连接电源后同步 |
| 离线 | 单机隔离 | 仅手动备份，适合安全环境 |

## 布局与主题
- 采用栅格化布局工具：拖拽窗口，即可吸附成分栏或浮层。
- 主题与字体遵循系统设置，可对单 Workspace 启用高对比或沉浸模式。

## 配置迁移
- Workspace 会自动备份到 `~/Tuff/workspaces`。
- 可导出加密档案，与团队成员共享并一键导入。

## 小贴士
- 使用「聚焦模式」隐藏一切次要插件，让屏幕回到任务主体。
- 将常用 Workflow 置顶在 Workspace 工具栏，开机后无需再手动触发。
