# Marketplace

## Discovery
- Home view splits into Featured, New, and Trending; every entry passes automated security scans.
- Natural language search (“organize screenshots”) suggests curated bundles.

## Detail Page
| Block | Content |
| --- | --- |
| Header | Name, developer, rating, workspace tags |
| Feature story | Capabilities, input types, gallery |
| Permissions | Clipboard, network, storage, etc. |
| Changelog | Semantic versions with diff links |

## Install Flow
1. Click Install.
2. Review permissions, confirm, and let hot reload finish.
3. Choose “Try now” or “Assign to workspace”.

## Ratings & Feedback
- Submit scores and comments in Settings → Plugins.
- Attach log snippets for faster responses.

## Enterprise Distribution
- Upload private, signed builds visible only to your org.
- Control auto-update channels per workspace fleet.
