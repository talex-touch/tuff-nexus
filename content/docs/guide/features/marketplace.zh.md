# 插件市场

## 发现
- 市场首页遵循「精选、最新、热门」三栏，所有条目均通过自动化安全扫描。
- 搜索栏支持自然语言：例如输入“整理截图”即可得到截图整理、OCR 等组合插件。

## 详情页结构
| 模块 | 内容 |
| --- | --- |
| 顶部 | 名称、开发者、评分、适用 Workspace |
| 功能段落 | 核心能力、支持的输入类型、示例截图 |
| 权限 | 读取剪贴板、网络访问等细粒度申明 |
| 更新日志 | 采用语义化版本，点击可拉起 diff |

## 安装体验
1. 选择插件后点击「安装」。
2. 显示所需权限，确认后自动下载并热加载。
3. 安装完成会弹出「立即体验」与「分配到 Workspace」两个按钮。

## 评分与反馈
- 在插件面板可直接提交评分与意见，开发者会收到通知。
- 反馈可附带日志片段，方便定位问题。

## 企业分发
- 支持上传私有市场包，统一签名后仅组织用户可见。
- 企业包可指定自动更新通道，确保版本一致。
