# Automation Control

## CoreBox Commands
| Command | Action |
| --- | --- |
| `open <app>` | Launch app or workspace |
| `run <workflow>` | Fire a workflow |
| `set focus` | Toggle Focus Mode |
| `diagnose` | Export diagnostics |
| `record macro` | Capture a macro |

## Shortcuts
- `⌘ + E` – summon CoreBox.
- `⌘ + Shift + K` – open Workflow Studio.
- `⌘ + Shift + Space` – floating widget board.
- `⌘ + Shift + M` – toggle Focus Mode.
- Remap everything per workspace in Settings.

## Clipboard Boosters
- Clipbox auto-detects text, image, files, and triggers workflows.
- Use Rules to clean Markdown, code, or spreadsheet data.
- Enable “Cross-device clipboard” for mobile parity.

## Automation Loop
1. Trigger via shortcut or clipboard.
2. Process via workflow or plugin scripts.
3. Present via widget, notification, or target app.
4. Review via `automation log`.

## Guidance
- Define the expected outcome first, then pick triggers.
- Require Touch ID / Windows Hello for sensitive commands.
