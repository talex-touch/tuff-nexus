# 效率面板

## Widget 组合
- 将计时器、任务列表、快捷命令放入同一面板，形成统一控制中心。
- Widget 支持拖拽排序与缩放，按住 `Option` 可对齐至网格。

## 模板策略
| 模板 | 适用场景 |
| --- | --- |
| 每日简报 | 展示今日任务、天气、会议 |
| 研发监控 | 接口延迟、构建状态、日志推送 |
| 创作仪表 | 灵感、草稿、发布计划 |

## 同步与共享
- Workspace 中的 Widget 布局与模板会自动同步。
- 可导出面板 JSON，与团队共享并快速导入。

## 第三方对接
- 使用 Web Widget 将 Figma、Notion 等面板嵌入。
- 通过 Channel API 拉取自定义数据，配合 Hooks 实时刷新。

## 维护建议
- 定期清理无关注项目的 Widget，保持专注。
- 在 Workflow 中设置「面板刷新」动作，确保数据最新。
