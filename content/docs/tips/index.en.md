# Usage Playbook

> Hidden tricks and high-level habits for a calm, Apple-like experience.

## Reading Order
1. **Intelligence Workflow** — build cross-plugin task chains.
2. **Automation Control** — command palette, shortcuts, clipboard flows.

## Shared Advice
- Trigger everything from CoreBox whenever possible.
- Replace manual steps with workflow templates and trace logs.
- Tune advanced settings per workspace; never force a single global profile.
