# FAQ

## Install
**Q: Installer won’t open.**  
A: Gatekeeper may block it—right-click Open once on macOS.

**Q: How do I join beta builds?**  
A: Settings → About → Channel → choose “Beta” and restart.

## Permissions
**Q: What permissions do plugins need?**  
A: Each plugin lists them during install; toggle them anytime in Settings → Plugins.

**Q: Clipboard access denied?**  
A: On macOS enable Tuff under System Settings → Privacy & Security → Input Monitoring.

## Performance
**Q: CoreBox feels slow.**  
A: Disable unused plugins or switch to on-demand loading.

**Q: Workflow stuck?**  
A: Open Workflow Studio → Live Log to find the slow node; split if needed.

## Account
**Q: Device limit?**  
A: Five by default; contact support or upgrade for more.

**Q: Delete my data?**  
A: Sign out, then choose “Remove local data”; cloud config clears within 24h.
