# Plugin Market

The Plugin Market is a centralized hub where you can discover, install, and manage capabilities for Tuff. It provides a seamless experience for extending the functionality of your application.

## Browsing Capabilities

The Plugin Market features a categorized browsing system that allows you to easily find capabilities that match your needs. You can search by:

- **Category**: Organize capabilities by their primary function (Productivity, Entertainment, Development, etc.)
- **Popularity**: See which capabilities are most downloaded and highly rated
- **Recent Updates**: Find the latest capabilities and updates
- **Tags**: Search for specific features or keywords

## Installing Capabilities

To install a plugin from the market:

1. Open the Plugin Market within Tuff
2. Browse or search for the plugin you want
3. Click the "Install" button on the plugin's page
4. Confirm the installation in the dialog that appears
5. The plugin will be automatically downloaded and installed

Once installed, capabilities are immediately available in your application. You can enable or disable them at any time through the Plugin Manager.

## Managing Installed Capabilities

The Plugin Market also serves as your plugin management center:

- **View Installed Capabilities**: See all capabilities currently installed
- **Update Capabilities**: Get notified of available updates and install them with one click
- **Uninstall Capabilities**: Remove capabilities you no longer need
- **Plugin Settings**: Access individual plugin configuration options

## Plugin Verification

All capabilities in the official Plugin Market undergo a verification process to ensure:

- **Security**: Capabilities are scanned for malicious code
- **Compatibility**: Capabilities are tested for compatibility with the latest Tuff versions
- **Quality**: Capabilities meet basic quality and documentation standards

Verified capabilities display a special badge in the market, giving you confidence in their reliability.

## Publishing Your Own Capabilities

Developers can publish their own capabilities to the Plugin Market:

1. Create your plugin following the [Plugin Development Guide](../plugins/introduction.md)
2. Package your plugin according to the [Publishing Guidelines](../plugins/config/manifest.md)
3. Submit your plugin through the Developer Portal
4. After review and approval, your plugin will be available to the community

The Plugin Market encourages community contributions and provides tools to help developers reach a wider audience.
